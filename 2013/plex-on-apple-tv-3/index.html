Plex on Apple TV 3

在要買 AppleTV 3 時就一直期待著，那天可以像 ATV 2 代一樣裝著 Plex player 觀看我電腦裡的影片，只是這天等了好久，一直沒實現，反到是新家買了三星的 TV ，裡面的 Samsung TV App store 居然有 Plex 可以裝! 讓我看影片很爽快! (大概只有這套 app 想裝，其它的都非常g肋，應該沒人想用 remote 選字上 youtube 吧!)

AppleTV 3 躺了好久，今天終於出頭天了! 有高手弄出了 PlexConnect，讓電腦當 server 透過 ATV3 改 dns 來連到電腦，看電腦上的影片，還免 jb 耶!! AWESOME!

重點不用1分鐘就搞定!! 真是無痛!

(那像之前連大陸地區的方法，麻煩又 lag 看得到吃不到 #$%^&* )

p.s. 我的電腦是 Mac OSX ，呃! 我不知 windows 是不是也可以1分鐘搞定啦~

###1. 如果電腦還沒有裝 Plex Media Server 的，要先裝一下!

###2. 在終端機 clone PlexConnect 的 repo. , 或是直接下載 Zip 檔案也行

###3. 編輯 Settings.py 檔案

! PlexConnect 的條件是需要 ATV 與電腦要在同一個網域下.

###4. 差不多了!
在電腦先開啟 Plex Media Server
接著在終端機下執行sudo ./PlexConnect.py 輸入密碼後，電腦端就搞定了，應該可以看到類似以下訊息

可以看到 Plex Connect 會連到 Plex Media Server，而且會佔用 80 port ， 因為我有開 apache 在 80 port , 所以要先關一下。

###5. ATV 上修改 DNS 設定
Plex Connect 的 Wiki 上有圖片教學如何設定 DNS

也就是在 設定 > 一般 > 選你連接網路 > 進入 DNS 修改成上面輸入的電腦區網 ip

接著進入 ATV 的預告片裡應該就會看到了!

只要在 Plex Media Server 設定好，中文字幕也沒問題! 但目前發現有部份1~2個中文字會有亂碼，其它都非常棒!

真的! 不用 1 分鐘就搞定了呢!!

設定

我自已字幕的設定是編碼要 UTF-8 比較不會有問題, 並在檔名在上加.zh movie_name.zh.str 這樣在字幕選擇時才會出現中文，不然會出現 unknown

分享我三星電視上的 Plex 其實可以設定的比較多，字幕大小等等，但重點是 ATV 不在無用武之地了!!!

Enjoy!!
