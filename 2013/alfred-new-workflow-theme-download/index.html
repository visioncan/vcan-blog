Alfred v2 新的擴充功能及主題

這次 v2 改版，最大的差異就在把原本在 v1 分開的 Shell Script , workflow, AppleScript … 整合成 workflows，來實作一下

來作一個簡單的取得 wifi 的 ip 的 workflow 吧!

####1.先在 workflows 標籤中下方按”+”

####2.出現如下3個流程，最左是設定要 keyboard search 的 “關鍵字”，中間是”指令動作”，最右是”結果通知”

####3.先設定 “關鍵字”，選 No Argument 是不帶入參數，可拉入 128 x 128 的 icon

####4.”指令動作”輸入，v2 版除了支援 bash 之外還支援 zsh, php, ruby, python, prel, osascript !

####5.”結果通知”可選 OSX 內建的 Notification Center 或 Growl

####6.最後完成結果!

只要打上 ip 便會在通知中出現。



藉由只打上2個字，就可以作完整個工作，實現快速化! 甚至比較復雜的指令、snippets、Bookmrk 等…工作都可以交給 Alfred 了~
