Alfred

Mac 上的聰明管家阿福 Alfred，升級v.2了

記得之前在 App store 上發現一個紫色的帽子，長居免費排行榜許久，那時還在1.0的版本，好奇驅使之下，載來玩玩，才發現真是超棒、幫助工作效率的軟體!! 讓我迅速地放棄掉了內建的 Spotlight!

Alfred 是 Mac 的管家阿福，讓我想到黑暗騎士的管家阿福 一樣好用!?， Alfred 是一個在工作列的軟體，啟用後是一個輸入框打上你要搜尋的字，可以快速啟用你搜尋的指令動作，在 App Store 上的免費版本只能做基本的軟體、網頁搜尋等。

對一般使用來說也許很夠用了，但購買了 powerpack (動力包!?) 後，可以把他由士兵變成美國隊長!!!(笑)
支援自訂主題、執行指令、還有很多加速生產力的功能!

因為只要輸入1~2個字，Alfred 就會列出軟體或指令列表，所以我的 Dock 變得如此清爽乾淨，開啟軟體都用有勞 Alfred 了。


Alfred 可以開發擴充套件，也可以用別人分享的擴充套件，支援加入結合 Shell語法、Applescript、workflow，簡化工作!

像是我要取得內外部 ip 位址，便會出現在系統通知

不要再每次都要開磁碟工具來修復權限了!

Alfred 已經深深植入我的手掌了~

最新 v2.0 正式版快釋出了，也馬上來用看看，而v1居然不能免費升級!!! 要免費升的要買 Mega 的 license 才可以，早知當初就買 Mega，現在要再花一次錢升級 …奧 但這次改進流程式的工作指令，變的更強大! 主題自訂自由度更高，這次心狠買了 Mega 了。

但因為工作流程的大改版，導至 v1 的擴充套件現在還沒辨法直接匯入到 v2，但官方有在 [blog 升級說明] (http://blog.alfredapp.com/2013/03/16/alfred-v2-importing-settings-from-alfred-v1/) 將在 v2.1 版本加入這個功能 (應該是付費的 App 更新挺快的)，期待快點啊!
不然 v2 一直沒辨法大力用，因為我大部份的擴充功能都在還是在 v1 上 ，還沒辨法換過去啊~~~~

v2.01 有出現了 import v1 按鈕，但… 只有些偏好設定轉移而已
